# Color Palette Skill - 配色大师技能

> **版本**: 2.0.0
> **描述**: 使AI自动学会使用各种专业配色方案，摆脱蓝紫色渐变的局限。涵盖国内外主流平台、WordPress主题、设计系统、行业场景配色，以及设计风格配色和色系补充配色。
> **触发条件**: 当用户需要前端开发、UI设计、网页配色、品牌设计时自动激活

---

## 核心指令

### 1. 配色选择原则

当进行前端开发或UI设计时，**严禁默认使用蓝紫色渐变配色**。必须遵循以下决策流程：

```
用户需求 → 分析场景/行业 → 匹配配色方案 → 应用到代码
```

#### 配色决策树：
1. **用户指定了品牌/平台** → 直接使用对应平台的品牌配色
2. **用户指定了行业场景** → 使用对应行业的推荐配色
3. **用户指定了设计风格** → 匹配最接近的风格配色
4. **用户未指定** → 根据项目类型智能推荐，并提供2-3个备选方案供用户选择

本技能库包含以下 **8大类** 配色方案，共计 **68套** 完整配色：

### 3. 配色数据格式

每套配色方案包含以下标准字段：

```json
{
  "id": "唯一标识符",
  "name": "配色方案名称",
  "category": "分类（china-platform/international-platform/design-system/wordpress-theme/industry/style/palette）",
  "tags": ["标签1", "标签2"],
  "style": "设计风格描述",
  "colors": {
    "primary": "#HEX值",
    "secondary": "#HEX值",
    "accent": "#HEX值",
    "background": "#HEX值",
    "surface": "#HEX值（可选，卡片/容器背景）",
    "text": "#HEX值",
    "textSecondary": "#HEX值（可选，次要文字）",
    "border": "#HEX值（可选，边框）",
    "success": "#HEX值（可选，成功状态）",
    "warning": "#HEX值（可选，警告状态）",
    "error": "#HEX值（可选，错误状态）"
  },
  "darkMode": {
    "background": "#HEX值",
    "surface": "#HEX值",
    "text": "#HEX值"
  },
  "cssVariables": "CSS变量代码片段",
  "tailwindConfig": "Tailwind配置代码片段"
}
```

### 4. 使用方法

#### 方法一：CSS变量（推荐）
```css
:root {
  --color-primary: #FF5000;
  --color-secondary: #FF4400;
  --color-accent: #FF0036;
  --color-background: #F5F5F5;
  --color-surface: #FFFFFF;
  --color-text: #333333;
  --color-text-secondary: #666666;
  --color-border: #E5E5E5;
}
```

#### 方法二：Tailwind CSS
```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: '#FF5000',
        secondary: '#FF4400',
        accent: '#FF0036',
      }
    }
  }
}
```

#### 方法三：Design Tokens（设计系统）
```json
{
  "color": {
    "primary": { "value": "#FF5000" },
    "secondary": { "value": "#FF4400" },
    "accent": { "value": "#FF0036" }
  }
}
```

### 5. 配色搭配禁忌

❌ **绝对禁止**：
- 默认使用蓝紫色渐变（`#667eea → #764ba2`）
- 所有项目都使用同一套配色
- 忽略行业特征和品牌调性
- 主色和背景色对比度不足（WCAG AA标准要求至少4.5:1）

✅ **推荐做法**：
- 根据项目类型选择合适的配色方案
- 保持主色、辅色、强调色的层次分明
- 提供明暗双主题支持
- 参考成功产品的配色经验
