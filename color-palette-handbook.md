- ❌ 默认使用蓝紫色渐变（`#667eea → #764ba2`）

- ❌ 主色和背景色对比度不足
- ❌ 忽略行业特征和品牌调性

- ❌ 同时使用超过3种高饱和度颜色

```css
:root {
  /* 核心色 */
  --color-primary: #FF5000;
  --color-secondary: #FF4400;

  --color-background: #F5F5F5;
  --color-surface: #FFFFFF;

  /* 文字 */
  --color-text: #333333;
  --color-text-secondary: #999999;

  /* 边框 */
  --color-border: #E5E5E5;

  --color-success: #10B981;
  --color-warning: #F59E0B;

  --color-text-secondary: #999999;

}
```
